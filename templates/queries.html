Division Query   Athletes who competed in ALL Men's 100m games

Uses double-nested NOT EXISTS — finds athletes for whom no game exists that they have NOT participated in.

SELECT a.registration_number, a.first_name, a.last_name FROM athlete a WHERE NOT EXISTS ( SELECT g.game_number FROM game g WHERE g.format = '100m' AND g.gender_category = 'Male' AND NOT EXISTS ( SELECT 1 FROM athlete_participation ap WHERE ap.athlete_registration_number = a.registration_number AND ap.format = g.format AND ap.gender_category = g.gender_category AND ap.game_number = g.game_number ) ) AND a.sport = '100m Sprint'

Nested Subquery   Athletes scoring above their event's average

Correlated subquery: for each row, computes the average score for that (format, gender_category) group and compares.

SELECT a.first_name, a.last_name, ap.format, ap.score, (SELECT ROUND(AVG(ap2.score), 2) FROM athlete_participation ap2 WHERE ap2.format = ap.format AND ap2.gender_category = ap.gender_category) AS event_avg FROM athlete a JOIN athlete_participation ap ON a.registration_number = ap.athlete_registration_number WHERE ap.score > ( SELECT AVG(ap2.score) FROM athlete_participation ap2 WHERE ap2.format = ap.format AND ap2.gender_category = ap.gender_category )

Aggregation + GROUP BY   Medal count grouped by sport

Uses SUM() with boolean expressions and COUNT(), grouped by sport.

SELECT e.sport, SUM(ap.medal = 'Gold') AS gold, SUM(ap.medal = 'Silver') AS silver, SUM(ap.medal = 'Bronze') AS bronze, COUNT(ap.medal) AS total FROM athlete_participation ap JOIN event e ON ap.format = e.format AND ap.gender_category = e.gender_category WHERE ap.medal IS NOT NULL GROUP BY e.sport ORDER BY total DESC
